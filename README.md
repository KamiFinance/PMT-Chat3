# PMT-Chat3

A decentralized blockchain messenger built with React + TypeScript + Vite.

## Features
- 🔐 Self-custody wallets (create, import, MetaMask)
- 💬 Real-time messaging via localStorage inbox polling
- 🎙 Voice messages with Pinata IPFS storage
- 🖼 Image & file sharing with IPFS
- 🤖 AI Assistant powered by Claude
- 🌙 Dark/Light mode
- 📱 Mobile responsive

## Setup
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Stack
- React 18 + TypeScript
- Vite 8
- PMT Chain (localStorage-based blockchain)
- Pinata IPFS for media storage
- Anthropic Claude AI
